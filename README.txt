Modificación de estructura de las instancias:
	- Razonamiento: No había una estructura regular o constante en relación a los espacios.
	
	- Cambios:
		- Se eliminaron las líneas vacías entre cada elemento.
		- Los tres componentes se movieron a una sola línea cada uno.
		- Las matrices se escribieron como una lista separado por comas "," (para cada valor de una fila) y con punto y comas ";" (para cada fila)

	- Ejemplo: En el siguiente caso, se cambió de

		opponents= [
		[3 4 -1 -2]
		[2 -1 4 -3]
		[4 -3 2 -1]
		[-3 -4 1 2]
		[-2 1 -4 3]
		[-4 3 -2 1]
		];

	a:
		opponents=3,4,-1,-2;2,-1,4,-3;4,-3,2,-1;-3,-4,1,2;-2,1,-4,3;-4,3,-2,1
